### Array in MATLAB
```MATLAB
a = [1 2 3];  %row vector
b = [1 2 3]';  % or b = [1; 2; 3]  % column vector
A = [1 2 3; 4 5 6];  % matrix
```
> ***MATLAB start from `1` NOT `0`***
